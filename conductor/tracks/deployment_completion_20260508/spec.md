# Specification: Finish Project Deployment (Hybrid)

## Overview
Complete the deployment of the Winery project: Frontend to **Vercel** and Backend (Server) to **Render**. Database hosted on **Neon**.

## Functional Requirements
- Deploy React Frontend to Vercel.
- Deploy Elysia Backend to Render.
- Connect Render Backend to Neon DB.
- Connect Vercel Frontend to Render Backend API.
- Set up CI/CD for both platforms via GitHub Actions.

## Acceptance Criteria
- [ ] Backend live on Render, connected to Neon.
- [ ] Frontend live on Vercel, connected to Render.
- [ ] Automatic deployments on push.
