# PB138 – Seminar 09: Auth & Observability

A TypeScript monorepo with an **Elysia** REST API and a **React** frontend using **TanStack Start** (SPA mode). This seminar adds authentication (Clerk), authorization (permissions-based RBAC), and observability (OpenTelemetry + Sentry → ClickStack).

## Prerequisites

- **Bun** — install from https://bun.sh
- **Docker** — for running PostgreSQL and ClickStack

## Getting started

```bash
# 1. Install dependencies
bun i

# 2. Start PostgreSQL + ClickStack
docker compose up -d

# 3. Copy environment variables and fill in Clerk + Sentry keys
cp apps/server/.env.example apps/server/.env
cp apps/web/.env.example apps/web/.env

# 4. Apply migrations & seed the database
bun run db:push
bun run db:seed

# 5. Start the server + frontend
bun run dev
```

| URL                            | What                           |
| ------------------------------ | ------------------------------ |
| http://localhost:3000          | REST API                       |
| http://localhost:3000/api-docs | Scalar API docs                |
| http://localhost:5173          | Frontend (TanStack Start SPA)  |
| http://localhost:8080          | ClickStack observability UI    |
