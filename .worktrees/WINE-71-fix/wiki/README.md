## Structure

Each wiki page is organized by:
- **Key concepts** — fundamental ideas and terminology
- **Code examples** — practical, runnable snippets
- **Patterns** — common approaches and best practices
- **Cross-references** — links to related pages within the wiki

## Usage

- Use these pages as **quick lookup references** while coding
- Link them in code reviews and documentation
- Cross-reference related concepts using internal links
- Keep examples concise but complete
