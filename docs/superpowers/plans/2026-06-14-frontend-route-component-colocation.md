# Frontend Route & Component Colocation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert flat dot-separated route files to a nested directory tree, then colocate every custom component (and its unit test) into the deepest `-components/` folder of the route subtree that uses it — leaving only `ui/`, `primitives/`, `layout/` in `components/`.

**Architecture:** Two phases on one branch. Phase A is a behavior-preserving route rename driven by a ts-morph codemod (TanStack treats `.` and `/` identically, so the route tree is unchanged). Phase B uses ts-morph's reverse-dependency graph to compute each component's deepest-common-ancestor route folder, then moves files and rewrites all imports project-wide. `bun run validate` is the acceptance gate.

**Tech Stack:** Bun, ts-morph (codemods), TanStack Router (Vite plugin), Vitest, Biome, TypeScript. All work is under `apps/web/`.

**Spec:** `docs/superpowers/specs/2026-06-14-frontend-route-component-colocation-design.md`

**Commit style:** Conventional commits, single-line subjects, scope `WINE-260`. Commit in reasonable chunks (Phase A as one commit; Phase B sliced per source feature folder).

## PHASE A — Route files: dot → directory

### Task A1: Write the route-conversion codemod

The transform for a top-level route file: take the basename without `.tsx`, split on `.`, join with `/`, re-add `.tsx`. Files whose basename has no extra `.` (e.g. `index.tsx`, `cart.tsx`, `__root.tsx`) are unchanged. Layout files convert correctly too (`_authenticated._admin.tsx` → `_authenticated/_admin.tsx`). ts-morph's `.move()` rewrites the moved file's own relative imports and every project reference automatically.

**Files:**
- Create: `apps/web/scripts/refactor/convert-routes.ts`

- [ ] **Step 1: Write the codemod**

```ts
// apps/web/scripts/refactor/convert-routes.ts
import { Project } from "ts-morph";
import path from "node:path";

const project = new Project({ tsConfigFilePath: "tsconfig.app.json" });
const routesDir = path.resolve("src/routes");

// Only top-level files directly in src/routes/. Subdirectories are already nested.
const moves: Array<{ from: string; to: string }> = [];
for (const sf of project.getSourceFiles("src/routes/*.tsx")) {
  const dir = sf.getDirectoryPath();
  if (path.resolve(dir) !== routesDir) continue; // skip files already in subdirs
  const base = sf.getBaseNameWithoutExtension(); // e.g. "shops.$id.inventory.new"
  if (!base.includes(".")) continue;             // index, cart, search, __root — leave
  const nested = base.split(".").join("/") + ".tsx";
  const to = path.join(routesDir, nested);
  moves.push({ from: sf.getFilePath(), to });
  sf.move(to);
}

await project.save();
console.log(`Moved ${moves.length} route files:`);
for (const m of moves) {
  console.log(`  ${path.relative(routesDir, m.from)} -> ${path.relative(routesDir, m.to)}`);
}
```

```ts
// apps/web/scripts/refactor/debarrel-forms.ts
import { Project } from "ts-morph";

const project = new Project({ tsConfigFilePath: "tsconfig.app.json" });
const map: Record<string, string> = {
  AddressFields: "@/components/forms/AddressFields",
  FormSection: "@/components/forms/FormSection",
  ImageUploadField: "@/components/forms/ImageUploadField",
  SubmitButton: "@/components/forms/SubmitButton",
  TextareaField: "@/components/forms/TextareaField",
  TextField: "@/components/forms/TextField",
};

for (const sf of project.getSourceFiles()) {
  const decl = sf.getImportDeclaration((d) => d.getModuleSpecifierValue() === "@/components/forms");
  if (!decl) continue;
  const named = decl.getNamedImports().map((n) => n.getName());
  decl.remove();
  for (const name of named) {
    sf.addImportDeclaration({ moduleSpecifier: map[name], namedImports: [name] });
  }
}
await project.save();
console.log("De-barreled @/components/forms");
```

### Task B2: Write the colocation computer and the mover

**Files:**
- Create: `apps/web/scripts/refactor/compute-colocation.ts`
- Create: `apps/web/scripts/refactor/apply-moves.ts`

- [ ] **Step 1: Write the placement computer**

Computes, for each in-scope component, the deepest route directory that is a common ancestor of every route file transitively importing it, then writes `moves.json`. Tests (`*.test.tsx`) follow the component they test.

```ts
// apps/web/scripts/refactor/compute-colocation.ts
import { Project, SourceFile } from "ts-morph";
import path from "node:path";
import { writeFileSync } from "node:fs";

const project = new Project({ tsConfigFilePath: "tsconfig.app.json" });
const SRC = path.resolve("src").replace(/\\/g, "/");

const IN_SCOPE = [
  "components/catalog", "components/shops", "components/events",
  "components/dashboard", "components/forms", "components/home",
  "components/reviews", "components/settings", "components/stats",
  "components/dev", "routes/-components",
];
const rel = (p: string) => path.relative(SRC, p.replace(/\\/g, "/")).replace(/\\/g, "/");
const inScope = (p: string) => IN_SCOPE.some((d) => rel(p).startsWith(d + "/"));

// A "route file" = under routes/, not inside any "-" prefixed segment (those are ignored by TanStack).
const isRouteFile = (p: string) => {
  const r = rel(p);
  if (!r.startsWith("routes/")) return false;
  return !r.split("/").some((seg) => seg.startsWith("-"));
};

const stripTest = (p: string) => p.replace(/\.test\.tsx$/, ".tsx");

// Memoized: set of route-file paths that transitively import `file`.
const cache = new Map<string, Set<string>>();
function routeUsers(file: SourceFile, seen = new Set<string>()): Set<string> {
  const key = file.getFilePath();
  if (cache.has(key)) return cache.get(key)!;
  if (seen.has(key)) return new Set();
  seen.add(key);
  const out = new Set<string>();
  for (const ref of file.getReferencingSourceFiles()) {
    const rp = ref.getFilePath();
    if (isRouteFile(rp)) out.add(rp);
    else if (inScope(rp)) for (const u of routeUsers(ref, seen)) out.add(u);
    // referenced by non-route, non-in-scope (e.g. components/layout) -> treat that as a
    // root-level user so the component stays broadly shared:
    else out.add("ROOT");
  }
  cache.set(key, out);
  return out;
}

function commonAncestorDir(routeFiles: string[]): string {
  if (routeFiles.includes("ROOT") || routeFiles.length === 0) return "routes";
  const dirs = routeFiles.map((f) => path.dirname(rel(f)).split("/"));
  const first = dirs[0];
  let i = 0;
  for (; i < first.length; i++) {
    if (!dirs.every((d) => d[i] === first[i])) break;
  }
  const ancestor = first.slice(0, i).join("/");
  return ancestor || "routes";
}

const moves: Array<{ from: string; to: string }> = [];
for (const sf of project.getSourceFiles()) {
  const p = sf.getFilePath();
  if (!inScope(p)) continue;
  const r = rel(p);
  const isTest = r.endsWith(".test.tsx");
  // For a test, anchor placement to the component under test.
  let users: Set<string>;
  if (isTest) {
    const target = project.getSourceFile(path.join(path.dirname(p), path.basename(stripTest(p))));
    users = target ? routeUsers(target) : routeUsers(sf);
  } else {
    users = routeUsers(sf);
  }
  const ancestor = commonAncestorDir([...users]); // e.g. "routes/shops/$id"
  const dest = `${ancestor.startsWith("routes") ? ancestor : "routes/" + ancestor}/-components/${path.basename(p)}`;
  if (rel(p) === dest) continue;
  moves.push({ from: r, to: dest });
}

moves.sort((a, b) => a.from.localeCompare(b.from));
writeFileSync("scripts/refactor/moves.json", JSON.stringify(moves, null, 2));
console.log(`Computed ${moves.length} moves -> scripts/refactor/moves.json`);
for (const m of moves) console.log(`  ${m.from}  ->  ${m.to}`);
```

- [ ] **Step 2: Write the generic mover (reused, supports per-folder slicing)**

```ts
// apps/web/scripts/refactor/apply-moves.ts
import { Project } from "ts-morph";
import { readFileSync } from "node:fs";
import path from "node:path";

const filter = process.argv[2] ?? ""; // optional: only apply moves whose `from` startsWith filter
const project = new Project({ tsConfigFilePath: "tsconfig.app.json" });
const moves: Array<{ from: string; to: string }> = JSON.parse(
  readFileSync("scripts/refactor/moves.json", "utf8"),
);

let n = 0;
for (const m of moves) {
  if (filter && !m.from.startsWith(filter)) continue;
  const sf = project.getSourceFile(path.resolve("src", m.from));
  if (!sf) { console.warn(`skip (already moved?): ${m.from}`); continue; }
  sf.move(path.resolve("src", m.to));
  n++;
}
await project.save();
console.log(`Applied ${n} moves${filter ? ` (filter: ${filter})` : ""}`);
```
